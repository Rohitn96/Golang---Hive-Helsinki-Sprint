# 🐝 Hive Helsinki — Go Sprint

Code written during the **Hive Helsinki** coding sprint — a peer-to-peer, project-based learning programme. This sprint focused on learning **Go (Golang)** through progressively challenging exercises and real tool-building projects.

---

## 🛠️ Tools Built

### 📝 NotesTool
A command-line notes management application written in Go.  
Supports creating, viewing, and managing notes directly from the terminal.

---

### 🌤️ WeatherStation
A Go application that fetches and displays weather data.  
Built to practise working with external APIs, JSON parsing, and Go modules.

---

### 🔐 Cypher Tool
An encryption/decryption utility written in Go.  
Explores string manipulation, byte operations, and basic cryptography concepts.

---

## 📂 Exercises (~90+ programs)

A structured series of Go programs covering core language concepts:

| Category | Topics |
|---|---|
| **Strings** | Reverse, capitalise, uppercase, split, whitespace handling |
| **Math** | Factorial (iterative & recursive), power functions |
| **Arrays & Slices** | Sorting, combinations, range removal |
| **Type Conversion** | String-to-int, bulk `atoi` |
| **Shell Scripting** | Loops, file operations, search utilities |

---

## 🔧 Shell Scripts
Supplementary bash scripts for file management and iteration exercises:  
`count.sh` · `loop.sh` · `find.sh` · `files.sh` · `introduction.sh`
